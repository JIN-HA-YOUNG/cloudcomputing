Q1.  숫자 맞추기 게임 
    - 1~100 사이의 임의 숫자를 발생 (random 모듈 이용)
    - 사용자로부터 1~100 사이의 임의 숫자를 입력 (input 함수)
    - 사용자가 입력한 데이터가 random하게 생성된 숫자가 맞는 지 판단 
    - 틀렸을 경우, 다시 사용자로부터 입력 
        - 입력받은 데이터가 컴퓨터가 생성한 데이터보다 작을 경우 '보다 큰 수를 입력' 출력
        - 입력받은 데이터가 컴퓨터가 생성한 데이터보다 큰 경우 '보다 작은 수를 입력' 출력
    - 정답을 맞출 때 까지 반복하며, 정답을 맞췄을 경우 몇번 만에 맞췄는지 출력 
        - 1~5번 사이면 '천재' 라고 출력
        - 5~10번 사이면 '보통' 라고 출력
        - 10번 이상이면 '바보' 라고 출력 

    - random, input, if ~ else, while, for 구문을 사용
